关于hexo博客的搭建

1.前置准备

1.1github

首先，需要自己注册一个github账号

然后，点击右上角的加号新建一个项目New repository

之后，在新建的选项里面，选择[^repository name]为×××.github.io，下面选择Public，然后Create repository

Check:显示Quick Setup等空白选项时表示成功

1.2git

点击下载安装git for windows(仅适用windows)

Check:打开cmd输入git --version若显示版本号即为成功

1.3node.js

点击选择左边LTS版本下载安装

Check:打开cmd输入node -v以及npm -v

1.4配置hexo

首先任选一个作为项目地址的位置新建文件夹作为站点目录，打开后右键选择Git Bash Here

输入npm install -g hexo，正常没问题，如果被墙又不能自行科学上网，就输入npm install -g cnpm --registry=https://registry.npm.taobao.org切换到淘宝镜像后输入cnpm install -g hexo

Check:输入hexo -v出现版本号即为成功

1.5自备:一个能写markdown文件的IDE(推荐用Microsoft配套的VSCode)，基本的VC框架(一般电脑都自带)

2.基本配置

2.1初始化

同样在git中输入hexo init,等待安装后会出现:

  • node_modules:存放插件
  • scaffolds:文章模板
  • source:所有页面的markdown/html代码
  • themes:下载的主题
  • _config.yml:站点配置文件
  • package.json:插件目录

2.2格式化

输入npm intall来下载一些基本包

4.自添加插件

hexo所提供的插件功能plugins种类非常的多，这一部分为附加部分
这里参考一个live2D插件的添加，其他插件请自行按需添加

5.推送po文

blog的目的在于推送po文，我们有了github提供的免费地址，需要通过GitHub的ssh将本地传到互联网上

在Git Bash中输入ssh-keygen -t rsa -C {email}，请把{email}替换为注册github所用的email，之后一路enter，就把ssh key存到电脑默认工作路径(一般为C:/Users/Administrator/.ssh)

打开该地址，找到一个id_rsa.pub文件，以记事本打开，复制内容，在github网站登陆后点击自己头像，点击settings在左边的SSH and GPG keys里点击New SSH key,将整个复制的内容粘贴

Check:在本地的Git Bash中输入ssh -T git@github.com,若提示Hi {username}! You've successfully authenticated, but GitHub does not provide shell access.则配置成功，并在站点目录中可以看到多了一个deploy文件夹

每次更新内容后发布:

Check:若发布成功，可以看到INFO Deploy done: git的提示，并且过一会打开网址能看到内容更新(有时需要等很久)
